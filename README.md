# 🛍️ Digitum Shops

Welcome to the official shop system of **The Republic of Digitum**.

Shops allow citizens to sell:
- recipes
- crafts
- ideas
- programs
- PCGs
- services
- techniques
- anything creative

Shops are fun, community-driven, and essential for circulating D$ in the economy.

All purchases must follow the PR-based process described in this repo.
